# LuoS
Soul App增强助手

## 功能
- 语音匹配直接查看对面身份
- 去开屏广告
- 去广场广告
- 去瞬间评论区广告

## 适配版本
4.67.0

## 使用说明
1. 下载安装本模块
2. 在LSPosed中激活本模块
3. 打开本模块进行设置
4. 重启Soul App
5. Enjoy
